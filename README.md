# Weather-Forecast-Web-App
Real-time weather application built with HTML, CSS, and JavaScript using API integration.

## Features
- Search weather by city name
- Displays temperature, weather condition, date & time
- Real-time data using WeatherAPI
- Responsive and modern UI

## Tech Stack
- HTML
- CSS
- JavaScript
- WeatherAPI

## How it Works
- Fetches live weather data using Fetch API
- Updates DOM dynamically
- Handles invalid city search gracefully
